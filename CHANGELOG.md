# Changelog

## 0.1.1

- Added a custom logo icon for the extension.
- Fixed a bug where binary files/images (e.g., untracked images) were read as text during diff collection, causing excessive token counts and API status 400 errors.

## 0.1.0

- Initial release of Git Comment:
  - Support for AI-generated Git commit messages.
  - Conventional Commits formatting.
  - Interactive settings dashboard.
  - Secure credential storage.
  - Multiple cloud and local providers (Gemini, OpenAI, Anthropic, Groq, DeepSeek, OpenRouter, Ollama).
